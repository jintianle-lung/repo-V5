# Data

This release includes the full tactile CSV dataset used by the R5 reproducibility package.

- `data/raw/` contains 126 tactile recordings organized by physical size and depth.
- `data/labels/` contains the active manual window labels for File 1, File 2, and File 3.
- File 1 and File 2 are used for training/validation splits; File 3 is the held-out test file.

The data are provided for research reproducibility of the submitted tactile nodule localization algorithm.
